#+title: Sky Emacs
#+startup: overview

* Purpose
This is the clean personal Emacs configuration experiment.

This is the active MSI Emacs config. Doom is frozen as reference material under
=~/.config/doom=.

* Ownership
#+begin_src text
Nix owns external tools, language servers, formatters, fonts, services, wrappers, and links.
Emacs owns Emacs packages, editor behavior, keymaps, completion, LSP client behavior, Org, Dired, terminal, and theme faces.
Projects own project-local metadata such as .dir-locals.el and language-server hints.
#+end_src

* Commands
After Home Manager links this directory to =~/.config/emacs=:

#+begin_src sh
emacs-sync
emacs
#+end_src

Doom is frozen as reference material under =~/.config/doom=.

* Runtime State
Runtime/package state belongs outside the repo:

#+begin_src text
~/.cache/emacs/
#+end_src

The source tree should stay limited to hand-written config files.
