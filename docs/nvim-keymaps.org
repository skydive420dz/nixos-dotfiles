* Neovim Keymaps
:PROPERTIES:
:CUSTOM_ID: neovim-keymaps
:END:
This is the user-facing keymap reference for the NixOS NVF Neovim profile.

=<leader>= is =<Space>=.

Generated against the staged NixOS NVF config on 2026-05-27. Plugin-internal =<Plug>= mappings are intentionally omitted unless they have a direct user-facing binding.

** Quick Anchors
:PROPERTIES:
:CUSTOM_ID: quick-anchors
:END:
| Key                       | Action                         | Use                                                |
|---------------------------+--------------------------------+----------------------------------------------------|
| =<leader>e=               | Open MiniFiles at current file | Browse beside the file you are editing.            |
| =<leader>E=               | Open MiniFiles at cwd          | Browse from the project/current working directory. |
| =<leader>ff=              | Telescope find files           | Jump to a file by name.                            |
| =<leader>fg=              | Telescope live grep            | Search text across the project.                    |
| =<leader>lf=              | Format buffer                  | Format with Conform, falling back to LSP.          |
| =<leader>tt=              | Toggle terminal                | Open/close the default ToggleTerm terminal.        |
| =<C-\>= in terminal mode  | Exit terminal mode             | Return from terminal input to normal mode.         |
| =<C-j/k/l>= in completion | Next / previous / accept       | Navigate and accept Blink completion items.        |
| =<Tab>= / =<S-Tab>=       | Next / previous buffer         | Cycle BufferLine buffers in normal mode.           |
| =<leader>w…=              | Window actions                 | Split, close, isolate, or equalize windows.        |
| =<leader>q…=              | Quit actions                   | Quit one, all, save-and-quit, or force quit.       |
| =<C-h/j/k/l>=             | Focus pane/window              | Move between Neovim splits or tmux panes.          |
| =<Arrow keys>=            | Resize pane/window             | Resize Neovim splits or tmux panes.                |
| =gra=                     | Code action                    | Show available LSP actions at cursor.              |
| =grn=                     | Rename symbol                  | Rename the symbol under cursor via LSP.            |
