* Contract And Fix
The controlling contract is
[[file:../audits/2026-07-19-quickshell-slice1-remainder.org][the Slice 1
remainder audit]], section =Slice 1E - Overlay Reopen Timer Cancellation=.

Launcher and clipboard =close()= start a 150 ms timer so their fade can finish
before transient state is cleared.  Before this slice, =show()= did not stop a
timer left by an interrupted close.  Reopening an overlay quickly could
therefore let the old timer clear the newly entered query or preview state.

The accepted payload adds only one statement to each affected =show()=:

#+begin_src qml
closeTimer.stop();
#+end_src

Affected paths:

- =config/quickshell/modules/launcher/LauncherView.qml=
- =config/quickshell/modules/clipboard/ClipboardView.qml=

The normal =close()= path still calls =closeTimer.restart()=.  Slice 1E does
not change animation timing, input bindings, clipboard contents, preview
publication, deletion logic, or the NixOS configuration.
