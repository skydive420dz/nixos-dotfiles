** Rhythm
:PROPERTIES:
:CUSTOM_ID: rhythm
:END:
- Work in small checkpoints.
- Keep clean git state between jobs.
- Sync live machine before testing when needed.
- Commit only after the live behavior is accepted.

** Commands
:PROPERTIES:
:CUSTOM_ID: commands
:END:
- =git status --short --branch=
- =git diff --stat=
- =git log --oneline -5=
- =git push=
